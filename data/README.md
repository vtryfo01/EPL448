# Data Directory

Place `dielectron.csv` here before running the notebooks.

## Download

The dataset is publicly available on Kaggle:

> **CERN Electron Collision Data**  
> https://www.kaggle.com/datasets/fedesoriano/cern-electron-collision-data

Alternatively, download from the CERN Open Data Portal:

> McCauley, T. (2014). *Events with two electrons from 2010*.  
> CERN Open Data Portal. DOI: 10.7483/OPENDATA.CMS.PCSW.AHVG

After downloading, place the file at:

```
data/dielectron.csv
```

The CSV file (~14.7 MB, 100,000 rows) is excluded from version control via
`.gitignore` to keep the repository lean and clone times fast.
